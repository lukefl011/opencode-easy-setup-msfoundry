# Context Profile: General

Use this profile for mixed software engineering tasks across app, services, scripts, and tooling.

## Objectives

- Deliver maintainable code with clear validation steps.
- Keep architecture decisions explicit and pragmatic.
- Prefer reusable abstractions over one-off hacks.

## Planning expectations

- include impact map across packages
- include API/schema considerations when relevant
- include tests and rollout risks

## Build/check expectations

- lint, typecheck, tests
- summarize blockers with concrete fix options

## Release expectations

- semantic version alignment
- changelog readiness
- env and secrets completeness
- rollback procedure
